# Your Turn! Exercise: Experimenting with Scaling, Load Testing, and Local High-availability Setup

<img src="https://media.giphy.com/media/13GIgrGdslD9oQ/giphy.gif" width=50%/>

The purpose of this exercise is that you experiment locally in virtual machines with scaling, load testing, and a local high-availability setup to prepare for your project work.

Clone the [exercise repository](https://github.com/itu-devops/scaling_exercises.git).

```bash
git clone https://github.com/itu-devops/scaling_exercises.git
```

First, follow the instructions in the [README.md file of branch `basic_scaling`](https://github.com/itu-devops/scaling_exercises/blob/basic_scaling/README.md) of the repository.
Thereafter, switch to the `main` branch, and work through the examples presented in its [README.md file](https://github.com/itu-devops/scaling_exercises/blob/main/README.md)
